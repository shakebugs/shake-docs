Whatever you tweak, your configuration settings apply to all Shake modules:

* [Crash reports](android/crash-reports/overview.md)

* [Feedback type](android/user-feedback/feedback-type.md) for User feedback (Crash reports don't have this element).
* [Email field](android/configuration-and-data/new-ticket-screen-elements.md)
* [Inspect button](android/configuration-and-data/new-ticket-screen-elements.md)
